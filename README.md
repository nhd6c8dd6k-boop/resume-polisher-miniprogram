# 简历润色微信小程序 MVP

这是一个可以直接导入微信开发者工具的简历润色小程序原型，适合用来演示“兼职/实习简历润色服务”。

目标很简单：先不要等完整上线，先拿这个小程序作为演示工具，卖一份 $5 CAD 的简历润色服务，做 4 单就是 $20 CAD。

## 功能

- 输入目标岗位
- 粘贴原简历内容
- 选择中文或英文
- 选择专业、简洁、有冲击力三种润色方向
- 生成润色结果并复制
- 页面内自带一个简单的变现话术

## 适合卖给谁

- 加拿大留学生
- 找兼职的人
- 找 Co-op / internship 的学生
- 英文简历不够自然的人
- 想把中文经历改成英文简历表达的人

## 使用方式

1. 打开微信开发者工具。
2. 选择导入项目。
3. 项目目录选择本文件夹。
4. AppID 可以先使用测试号或游客模式。

## 赚钱切入点

先不要卖“小程序”，先卖“简历润色服务”：

> 我可以帮你把兼职/实习简历润色成更专业的中英双语版本，前 10 位 $5 CAD/份，当天交付。

做 4 单就是 $20 CAD。这个小程序可以作为演示工具，也可以之后接入 AI API 做成自动化版本。

## 建议定价

- $5 CAD：单页简历快速润色
- $10 CAD：中英双语版本
- $15 CAD：简历 + 求职自我介绍
- $20 CAD：简历 + Cover Letter 简版

先从 $5 CAD 开始，重点是拿到第一单和第一条反馈。

## 交付流程

1. 对方发来简历文字或截图。
2. 询问目标岗位和语言。
3. 用小程序生成初稿。
4. 人工检查语气、岗位关键词和错别字。
5. 发回最终文本，可附带 3 条修改建议。

## GitHub 项目介绍

这是一个轻量级微信小程序 MVP，用于验证 AI 简历润色服务的需求。当前版本使用本地规则生成润色结果，方便快速演示；后续可以接入真实 AI API、支付和历史记录。
