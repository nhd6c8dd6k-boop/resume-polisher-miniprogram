## README 开头展示语

Resume Polisher Mini Program is a lightweight WeChat Mini Program prototype for helping students improve resume bullet points for part-time jobs, internships, and entry-level roles.

The current MVP uses local rule-based generation for fast demos. It can later be connected to an AI API, payment flow, upload parsing, and saved history.

## 置顶项目介绍

我会把这个仓库置顶在 GitHub，用来展示一个能快速验证商业需求的小产品：简历润色小程序。它不是只为了技术展示，而是为了测试一个很小但真实的付费场景：学生愿不愿意付 $5 CAD 得到一份更专业的兼职/实习简历。
